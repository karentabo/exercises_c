C Programming Exercises Portfolio

This repository showcases a series of C programming exercises demonstrating practical skills in problem-solving, algorithm design, and fundamental programming concepts.

Skills Demonstrated
- Core C programming (loops, conditionals, functions)
- Arrays, matrices, and basic data structures
- Mathematical and logical problem solving
- Input/output handling and program flow
- Writing clean, modular, and readable code

Key Exercises
Exercise	Focus
alturas.c	Height calculations and averages
crescente.c	Sorting values in ascending order
exercicio_baskara.c	Quadratic equation solver
exercicio_idades.c	Age analysis and average calculation
negativos.c	Handling negative numbers - Finding diagonal numbers
retangulo.c	Calculating area and perimeter
soma_impares.c	Summing odd numbers
tabuada.c	Generating multiplication tables
terreno.c	Land area calculation

Usage
Clone the repository:
git clone https://github.com/karentabo/exercises_c.git
cd exercises_c

Compile a program:
gcc filename.c -o filename

Run the program:
./filename

Purpose
This collection highlights the ability to:
- Implement algorithms efficiently
- Work with arrays and matrices
- Solve practical programming challenges
- Build a strong foundation in C for further software development
